# airplane_summerProject
Avião de controle remoto com uso de arduino uno e nano

### link do avião no wokwi
https://wokwi.com/projects/450889991064560641


Aeromodelo projetado por conta própria (Seguindo modelo CESSNA 180);
Materiais: 
 - Papelão
 - Isopor
 - Palito de churrasco

 (tanto no avião quanto no controle)

 ### Componentes 
 ## Controle: 
 - Arduino Uno;
 - Joysticks;
 - nRF24L01 with PA;
 - MB102 breadboard power (with an 9v Battery)

 ## Avião
- Arduino Nano;
- Servos
- nRF24L01 without PA;
- ESC 30A with an 5V/2A output;
- Li-Poly 11.1 V 1500 mAh 3 cells Batery;
- Brushless A2212 / 13T Motor 

O que foi usado em cada parte (controle e sistema do avião) está comentado nos commits de cada um.
